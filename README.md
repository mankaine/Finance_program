## DESCRIPTION
This program is intended to log and finance your budget. <br />
<br />
Features:<br />
Enter transactions in a shell user interface<br />
Import, edit, and view transactions<br />
View transactions<br />
Set budgets<br />
<br />
Written in Python.<br />
